Notes: Originally this was just proof of concept to get a feel for combigning a hexegonal map with an isometric overlay. This was done with pure svg as to experiment with utilizing svg primiatives for rendering.

Changes:
• Render SVG as hexes and isometric styles both that overlay seemlessly
• Added svg objects to be rendered
• Added edge shading to isometric tiles to blend with the hex underlaying map better
• Added height mapping to individual points in each hex
• Added heightable water tiles
• Added viewport with pan and zoom controls
• Converted height system to be 256 based so each block can be +/- 256 points. Generally you want to reduce a block, not add. Adding to the block is merely to add terrain bumps without adding another layer
• Added Layers to the hex grid, this also broke the trees and iso grid, to be fixed.
